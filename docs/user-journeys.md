# User Journeys

## First Run
- Download installer, launch, choose secondary monitor, enable always-on-top (optional).
- Pick starter layout (system health + clock + notes).
- Walkthrough to add/remove widgets.

## Power Saving / Ambient Mode
- Toggle power saving mode to dim/black most of the screen while keeping selected widgets visible.
- Choose which widgets stay visible (e.g., alerts, clock, one metric card) and set timeout to auto-enable.

## Customize Widgets
- Open settings panel, browse widget registry, drag to layout, resize.
- Adjust update intervals and themes.
- Add a shortcuts/launcher widget and pin favorite games, tools, or files.

## Alert Triage
- Alerts widget shows info/warning/critical with timestamps and channels.
- Snooze, acknowledge, or open settings to tune rules.

## Notifications Flow
- Ingest system/app/custom notifications via IPC or integrations.
- Integrate optional sources (Facebook, Messenger, WhatsApp, Slack, Discord) with per-source opt-in.
- Filter by priority, pin important items, summarize bursty feeds.
- Show pipeline/automation (e.g., n8n) status with up/down or last-run result.

## Maintenance
- Update app via auto-updater, review release notes.
- Export/import settings and alert rules.
