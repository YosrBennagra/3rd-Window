# Product Brief

## Problem
Users with multi-monitor setups want a dedicated, glanceable dashboard for system health, notifications, schedule, and quick notes without alt-tabbing.

## Audience
- Power users, engineers, gamers, creators with dual monitors.
- IT/ops needing persistent alert surface.

## Jobs to Be Done
- Monitor system vitals at a glance.
- Keep critical notifications visible and filtered by priority.
- Pin time/date, events, and quick notes on a side screen.
- Trigger alerts when thresholds are crossed.
- Launch frequently used games/apps/files from a shortcuts widget without leaving the dashboard.
- Keep select widgets visible in a low-power ambient mode.

## Success Metrics (examples)
- Time to first dashboard < 3 minutes.
- Widget add/remove < 10 seconds.
- CPU overhead idle < 2%, GPU overhead negligible.
- Alert false-positive rate < 5% for default rules.
- NPS/CSAT for customization > 8/10.

## Differentiators
- Always-on-top secondary-screen focus.
- Modular widget registry with alert engine baked in.
- Balanced visual density and glanceability.
- Power-saving ambient mode that blackens unused screen areas while keeping chosen widgets live.
- Shortcut/launcher widget acting like a lightweight secondary OS surface.
- Optional connectors for popular services (Facebook, Messenger, WhatsApp, Slack, Discord) and automation status (e.g., n8n health).
