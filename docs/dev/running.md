# Running (Draft)

- Dev: `npm run tauri:dev` (starts Vite on 5173 and launches Tauri shell).
- Build: `npm run tauri:build` (Tauri bundle).
- UI-only: `npm run dev:ui` / `npm run build:ui` if you want to run the web app without the shell.
- Tests: `npm run test` (unit), `npm run e2e` (E2E placeholder).
