# Roadmap (Draft)

## Milestone 1: Core Dashboard
- Layout grid, widget host, always-on-top toggle.
- Core widgets: clock/calendar, notes, RAM/disk/network basics.

## Milestone 2: Metrics Depth
- CPU/GPU temps, richer network stats, better formatting.
- Settings persistence, import/export.

## Milestone 3: Alerts Engine
- Rules, thresholds, schedules, alert channels (toast/sound/webhook), alert widget.

## Milestone 4: Notifications
- System/app/custom notifications ingestion, filtering, summarization options.

## Milestone 5: Polishing and Release
- Theming, accessibility pass, performance tuning, packaging/signing, store listing.
