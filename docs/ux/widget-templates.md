# Widget Templates

- Notifications: list with priority badges, actions (pin, dismiss).
- Temperature: CPU/GPU with gauges and thresholds.
- RAM Usage: bar/percentage, top processes (optional).
- Disk Usage: capacity, free space, per-disk list.
- Network Speed: up/down, latency; mini sparkline.
- Clock/Calendar: time, date, upcoming events slot.
- Notes: quick text, pinned notes.
- Alerts: current and history, severity filter.
- Shortcuts/Launcher: pinned games/apps/files with quick launch and optional icons.
- Integrations/Notifications: per-source opt-in tiles (Facebook, Messenger, WhatsApp, Slack, Discord) and pipeline status (e.g., n8n) showing up/down and last run.
